# Segments Guide

Segments are sections of trail where athletes compete for the best time. This guide covers how to find, create, and compete on segments.

## Finding Segments

### Segment List

1. Go to "Segments" in the main menu
2. Browse the list of all segments
3. Each entry shows:
   - Segment name
   - Distance and elevation
   - Activity type
   - Number of attempts

### Map View

1. Click "Map" to switch to map view
2. Segments appear as lines on the map
3. Zoom and pan to explore different areas
4. Click a segment line to see details

### Filters

Use filters to narrow down segments:

- **Activity Type**: Run, Ride, Hike, etc.
- **Search**: Find segments by name
- **Sort**: By popularity, distance, or recent

## Segment Details

Click on any segment to see:

### Overview
- Name and description
- Creator and creation date
- Activity type
- Distance and elevation gain

### Route
- Map showing the segment path
- Elevation profile
- Start and end markers

### Leaderboard
- Top performers
- Your personal record
- Filter by age/gender

### Your Efforts
- All your recorded times on this segment
- Date and activity link for each effort
- PR indicator for your best time

## Creating Segments

You can create segments from any of your activities.

### Step 1: Open an Activity

1. Go to your activity detail page
2. Scroll to the elevation profile

### Step 2: Select Start Point

1. Click "Create Segment" button
2. Click on the elevation profile to set the start point
3. A marker will appear on the map

### Step 3: Select End Point

1. Click again on the elevation profile to set the end point
2. The segment route will highlight on the map
3. Review the distance and elevation

### Step 4: Add Details

1. Enter a segment name
2. Add an optional description
3. Confirm the activity type

### Step 5: Submit

1. Click "Create Segment"
2. Your segment is now live and available for competition

### Tips for Good Segments

- **Clear Start/End**: Choose distinct, findable locations
- **Meaningful Challenge**: Not too short, not too long
- **Descriptive Name**: Include the trail name or location
- **One Direction**: Segments are directional - consider both ways

## Competing on Segments

### Automatic Matching

When you upload an activity, Track Leader automatically:
1. Scans your route against all segments
2. Records efforts for matching segments
3. Updates leaderboards

### Effort Recording

An effort is recorded when:
- Your track passes through the start point
- Your track passes through the end point
- The activity type matches the segment type
- You're within the matching tolerance (~50m)

### Personal Records

Your fastest time on a segment is marked as your PR (Personal Record). You can track:
- Current PR time
- PR date
- Improvement over time

## Crowns

Earn crowns by holding the top position on a segment:

| Crown | Description |
|-------|-------------|
| KOM | King of the Mountain - Fastest male overall |
| QOM | Queen of the Mountain - Fastest female overall |

### Crown Counts

Your profile shows:
- Total KOMs/QOMs held
- Recent crown changes
- Crown history

## Segment Hazards

### Flagged Segments

Segments can be flagged for:
- Safety concerns
- Inaccurate routing
- Private property

Flagged segments may be hidden or removed.

### GPS Issues

Sometimes segments don't match correctly due to:
- GPS drift
- Signal loss
- Device accuracy

If your effort isn't recorded, check your GPS track quality.

## Exploring Nearby

Use the map view to find segments near you:

1. Allow location access (optional)
2. The map centers on your location
3. Browse nearby segments
4. Plan your next challenge!

## Segment Etiquette

- **Respect Access**: Only create segments on legal trails
- **Be Accurate**: Use quality GPS data for creation
- **Name Clearly**: Help others find and identify segments
- **Report Issues**: Flag problematic segments
